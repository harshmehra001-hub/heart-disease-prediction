# heart-disease-prediction
"Machine Learning-based Heart Disease Prediction Web App built with Scikit-learn and Streamlit."

Heart Disease Prediction Web App

Overview

This project predicts the risk of heart disease using Machine Learning. Multiple Scikit-learn algorithms were tested, and the best-performing model was selected.

Features

- Heart disease prediction
- Streamlit web interface
- Data preprocessing and scaling
- Multiple model comparison
- Real-time prediction

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
- Matplotlib
- Seaborn
